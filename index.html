HOME

This is the main screen of Can or Not.

It helps people shape text into natural Singlish prompts or clearer ChatGPT-ready prompts without extra clutter.

The website is designed as a small retro system where each section feels contained and easy to explore.

HOW IT WORKS

You type your original idea into the input box.

The left side helps rewrite it into a more natural Singaporean Singlish style.

The right side improves the wording so it becomes a cleaner prompt to paste into ChatGPT.

EXAMPLES

Example one: turn a plain sentence into a more local and human Singlish phrasing.

Example two: turn a rough question into a clearer and more useful ChatGPT prompt.

The point is not only output, but helping the user shape intention before asking AI.

ABOUT

Can or Not is a lightweight prompt-shaping tool with local flavour.

It was built to make interaction with AI clearer, more direct, and more usable for everyday people.

The project combines retro interface design, practical prompt engineering, and a simple human-first workflow.
